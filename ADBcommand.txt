.\adb devices - отобразить все подключенные девайсы
.\adb install -f ‘путь к файлу apk’ - установка app (если без ‘-f’ - устанавливается на карту)
.\adb uninstall ‘путь к файлу apk’ - удаление app
.\adb uninstall ‘идентификатор apk’ - удаление app
.\adb shell ‘pm list packages -f’ | findstr название app - поиск идентификатора app (findstr)
.\adb shell ‘pm list packages -f | grep название app’ - поиск идентификатора app (grep)
.\adb logical -d | findstr идентификатор app - получить логи по идентификатору app 
.\adb logical -d | findstr идентификатор app > название файла.log- снять логи в файл
.\adb shell screencap /путь в телефоне, куда сохранится файл/название файла.png - сделать скриншот экрана телефона (сохраняет в память телефона)
.\adb pull /путь в телефоне, где сохранен скриншот/название файла.png название файла.png - сделать скриншот экрана телефона (сохраняет в память телефона)
.\adb push название файла на компьютере.png /путь в телефоне, куда будет сохранен этот файл/название файла.png - залить скриншот с компьютера в телефон
